# Multi-Sig-Wallet
Imagine a company has $1 million in crypto. If only ONE person controls it:  That person can steal it If that person dies or loses their phone — money is gone forever. Hackers only need to attack ONE person.  Multi-Sig solves this:  3 people are owners At least 2 must agree before ANY money moves Even if a hacker steals 1 person's account.
